\newcommand{\reflowpage}[3]{%
  \clearpage
  \section{#1}
  \markboth{#1}{#1}
  \begin{pageintrobox}
    {\small\scshape\color{accent} Source\par}
    {\normalsize #2\par}
    {\small\color{titleink!75}Reflowed from column-by-column OCR so the booklet reads like a typeset document rather than a scan transcript.}
  \end{pageintrobox}
  \begingroup
  \setstretch{1.06}
  \small
  \input{#3}
  \endgroup
}

\section*{About This Edition}
This edition uses OCR text extracted from the original scenario scans rather than embedding the scans themselves. The content is therefore searchable and copyable. Because it is scan-derived text, minor OCR artifacts may remain, especially in dense unit setup listings and column transitions.
